# 小红书短线策略回测

把评论区三条玩法收成可回测规则，用 baostock **未复权日线**（2024-01-02 ~ 2025-12-31，约 4200 只 A 股）实测。

> 这不是投资建议。结果高度依赖成交假设；原文「量比 20–60」是分时口径，日线无法直接复现。

## 策略定义

| ID | 原帖意图 | 回测规则 | 关键近似 |
|----|----------|----------|----------|
| A | 9:30 量比 20–60 + 热点板块 | 开盘涨幅 2%–7%，行业开盘涨幅 Top3，按量比取 Top3，次日开盘卖 | 日线量比≠分时量比；同日量比有前视 |
| B | 自选 5 只，开盘跌停条件单买入，反弹几个点卖 | 昨 5 日跌幅最差且行业分散的 5 只；开盘跌停买入；盘中 +3% 或次日开盘卖 | 「自选」主观 → 规则化池 |
| C | 涨停且资金净流入 >20 亿打板，次日有赚就跑 | 收盘涨停且成交额 ≥20 亿；涨停价买入；次日开盘卖 | **成交额 ≠ 净流入**；涨停成交偏乐观 |

费用假设：双边合计约 15bp（`FEE_ROUNDTRIP=0.0015`）。

## 如何跑

```bash
pip install -r requirements.txt

# 日线（全市场，约 30 分钟，8 workers）
python download_parallel.py --start 2024-01-01 --end 2025-12-31 --workers 8

# 可选：开盘 5 分钟量比（更接近原文量比）
python download_open5m.py --start 2024-01-01 --end 2025-12-31 --workers 8 --only-from-panel --max-stocks 1200

python run_backtest.py
```

数据缓存于 `data/`（bars 不入库）。结果在 `results/`。

## 回测结果（日线，2024–2025）

样本：`1,977,804` 行，`4229` 只股票，`485` 个交易日。

| 策略 | 笔数 | 胜率 | 平均单笔 | 盈亏比(PF) | 累计收益* | 最大回撤* |
|------|------|------|----------|------------|-----------|-----------|
| A 原文阈值 20–60 套日线 | 0 | — | — | — | 0% | 0% |
| A 日线放量代理 3–8 | 365 | 50.4% | +0.65% | 1.30 | +55.8% | **-67.1%** |
| A 无前视（昨量比+开盘） | 482 | 38.6% | **-0.47%** | 0.83 | **-84.9%** | -88.2% |
| B 规则化 5 自选跌停 | 4 | 75% | +1.27% | 2.47 | +1.0% | -0.7% |
| B 全部开盘跌停（对照） | 882 | 57.4% | **-1.74%** | 0.48 | **-95.9%** | -96.0% |
| C 打板乐观成交 | 934 | 53.4% | +0.75% | 1.63 | +789%† | -25.8% |
| C 打板保守（曾打开） | 934 | 53.4% | +0.75% | 1.63 | +789%† | -25.8% |

\*累计收益 / 回撤按「有信号日满仓等权、收益滚入本金」计算，会放大趋势，**优先看胜率 / 平均单笔 / 盈亏比**。  
†C 含 2024-09 行情脉冲；且用成交额代理净流入、假设涨停价能买到，**显著偏乐观**。乐观与保守结果相同，说明样本里几乎都能在盘中打开后封板。

### 读法

1. **原文量比 20–60 不能直接套日线**：全样本日线量比落在 20–60 的只有几十次，叠加开盘涨幅与热点后 **0 笔成交**。
2. **A 一旦去掉前视就变负期望**：无前视版胜率 39%、平均单笔 -0.47%，说明「看到放量再追」在日线代理下站不住。
3. **B 跌停抄底是接飞刀**：放宽到全部开盘跌停后，胜率虽过半，但亏的时候亏得更大（均亏 -7.9% vs 均盈 +2.8%），两年接近打穿。
4. **C 看起来最好，但假设最假**：净流入换成成交额、默认排板成功；真实打板成交率会低很多，收益应大幅打折。
