# 小红书短线策略回测

把评论区三条玩法收成可回测规则，用 baostock **未复权日线**（2024-01-02 ~ 2025-12-31，约 4200 只 A 股）实测；策略 A 额外用 **开盘 5 分钟量** 近似原文「09:30 量比」。

> 这不是投资建议。结果高度依赖成交假设与数据口径。

## 策略定义

| ID | 原帖意图 | 回测规则 | 关键近似 |
|----|----------|----------|----------|
| A | 9:30 量比 20–60 + 热点板块 | 开盘涨幅 2%–7%，行业开盘涨幅 Top3，按量比取 Top3，次日开盘卖 | 分时量比用首根 5 分钟成交量 / 近 5 日同期均量；日线量比另作对照 |
| B | 自选 5 只，开盘跌停条件单买入，反弹几个点卖 | 昨 5 日跌幅最差且行业分散的 5 只；开盘跌停买入；盘中 +3% 或次日开盘卖 | 「自选」主观 → 规则化池；另给「全部开盘跌停」对照 |
| C | 涨停且资金净流入 >20 亿打板，次日有赚就跑 | 收盘涨停且成交额 ≥20 亿；涨停价买入；次日开盘卖 | **成交额 ≠ 净流入**；涨停成交偏乐观 |

费用假设：双边合计约 15bp（`FEE_ROUNDTRIP=0.0015`）。

## 如何跑

```bash
pip install -r requirements.txt

# 日线（全市场，约 30 分钟，8 workers）
python download_parallel.py --start 2024-01-01 --end 2025-12-31 --workers 8

# 可选：开盘 5 分钟量比（更接近原文量比；本仓库实测抽了约 1100 只沪市主板）
python download_open5m.py --start 2024-01-01 --end 2025-12-31 --workers 8 --only-from-panel --max-stocks 1200

python run_backtest.py
```

数据缓存于 `data/`（bars / open5m / panel 不入库）。结果在 `results/`。

## 回测结果（2024–2025）

日线样本：`1,977,804` 行，`4229` 只股票，`485` 个交易日。  
开盘 5 分钟量比样本：约 `1104` 只（主要为 `sh.60` 主板）。

| 策略 | 笔数 | 胜率 | 平均单笔 | 盈亏比(PF) | 累计收益* | 最大回撤* |
|------|------|------|----------|------------|-----------|-----------|
| **A 开盘5分钟量比 20–60** | 26 | 46.2% | **+0.91%** | 1.34 | +2.1% | -13.3% |
| A 原文阈值 20–60 套日线 | 0 | — | — | — | 0% | 0% |
| A 日线放量代理 3–8 | 365 | 50.4% | +0.65% | 1.30 | +55.8% | **-67.1%** |
| A 无前视（昨量比+开盘） | 482 | 38.6% | **-0.47%** | 0.83 | **-84.9%** | -88.2% |
| B 规则化 5 自选跌停 | 4 | 75% | +1.27% | 2.47 | +1.0% | -0.7% |
| B 全部开盘跌停（对照） | 882 | 57.4% | **-1.74%** | 0.48 | **-95.9%** | -96.0% |
| C 打板乐观成交 | 934 | 53.4% | +0.75% | 1.63 | +789%† | -25.8% |
| C 打板保守（曾打开） | 934 | 53.4% | +0.75% | 1.63 | +789%† | -25.8% |

\*累计收益 / 回撤按「有信号日满仓等权、收益滚入本金」计算，会放大趋势，**优先看胜率 / 平均单笔 / 盈亏比**。  
†C 含 2024-09 行情脉冲；用成交额代理净流入、假设涨停价能买到，**显著偏乐观**。乐观与保守结果相同，说明样本里几乎都能在盘中打开后封板。

### 结论

1. **原文量比不能直接套日线**：日线量比落在 20–60 且叠加过滤后 **0 笔**。
2. **更接近原文的开盘 5 分钟量比**：在约 1100 只沪市主板里筛出 26 笔，平均单笔略正、样本很小，远不够当稳定 alpha。
3. **A 去掉前视就变负期望**：无前视版胜率 39%、平均单笔 -0.47%。
4. **B 跌停抄底是接飞刀**：放宽后胜率过半，但均亏 -7.9% vs 均盈 +2.8%，两年接近打穿。
5. **C 看起来最好，假设也最假**：真实排板成交率会低很多，收益应大幅打折。
